What to Download and from Where ?

1. Go to https://getbootstrap.com/docs/4.3/getting-started/download/ and Click on Download Button 

2. Download the zip 

3. Extract the zip

4. Then delete all the files from the CSS and JS except --> bootstrap.min --> in both folders

5. Then  create a file name inside JS folder -> popper.js

6. Here is the link of the file ->https://code.jquery.com/jquery-3.6.0.min.js

7. Then  create a file name inside JS folder -> jquery.js

8. Here is the link of the file ->https://code.jquery.com/jquery-3.3.1.slim.min.js

9. Things we Need to include using Bootstrap

   1. We need to inlude CSS_min_css after Title .
   2. We need to inlude JS/jquery in Body.
   3. We need to inlude JS/jpopper in Body.
   4. We need to inlude Bootstrap_min_JS in Body.
